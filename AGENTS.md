# AGENTS.md

## Contexto del proyecto
Este repositorio contiene un prototipo navegable en Java Swing para una demo académica.
No es un sistema final ni un producto real.

## Objetivo
Mostrar de forma visual y simple cómo trabajaría el rol Encargado en un sistema informatizado para un kiosco.

## Reglas
- Mantener el proyecto pequeño.
- No agregar frameworks.
- No agregar persistencia.
- No agregar base de datos.
- No agregar autenticación.
- No agregar lógica de negocio real.
- Priorizar claridad visual y facilidad de ejecución en NetBeans.
- Usar Java Swing estándar.
- Usar datos mock realistas.
- No sobreingenierizar.

## Pantallas requeridas
- Menú principal
- Apertura de cuenta corriente
- Registro de consumo personal
- Cierre de caja
- Consulta de productos consumidos y vencidos

## Criterios de éxito
- El proyecto compila.
- El proyecto se puede ejecutar fácilmente.
- La navegación funciona.
- La demo es entendible para usuarios no técnicos.
